# Windows客户端配置说明

## V2RayN使用指南

### 下载与安装
1. 访问 [V2RayN GitHub发布页](https://github.com/2dust/v2rayN/releases)
2. 下载最新版本的 `v2rayN-Core.zip` 文件
3. 解压到任意文件夹
4. 运行 `v2rayN.exe` 即可

### 添加服务器
1. 打开V2RayN软件
2. 右键点击系统托盘图标，选择「服务器」→「添加[VLESS]服务器」
3. 填写以下信息：
   - **地址**：服务器IP地址或域名
   - **端口**：服务器端口（默认443）
   - **ID**：UUID（从部署脚本输出中获取）
   - **流控**：无
   - **传输协议**：tcp
   - **伪装类型**：none
   - **TLS**：开启
   - **SNI**：填写伪装的域名（从部署脚本输出中获取）
   - **ALPN**：h2,http/1.1
   - **REALITY**：开启
   - **公钥**：从部署脚本输出中获取
   - **短ID**：从部署脚本输出中获取
4. 点击「确定」保存配置

### 连接VPN
1. 选择添加的服务器
2. 点击「启动」按钮连接
3. 系统托盘图标变为绿色表示连接成功

### 验证连接
1. 打开浏览器访问 [ipinfo.io](https://ipinfo.io)
2. 查看显示的IP地址是否为服务器IP

### 常见问题
- **连接失败**：检查服务器地址、端口、UUID等参数是否正确
- **速度慢**：尝试更换服务器或调整传输协议
- **无法访问某些网站**：检查代理规则设置，确保相关网站被正确代理
